M.S. Project

With the support of the CONAHCYT fellowship and ITSON, I developed a sap flow monitoring system for the quantification of transpiration in wheat plants. My responsibilities included sensor fabrication, isolation, data collection system design, and calibration. As a result, we created an automatic tool to obtain real-time transpiration, which addresses local challenges related to wheat crop production in the Yaqui Valley, Sonora, Mexico. Submitted manuscript, currently under review.
